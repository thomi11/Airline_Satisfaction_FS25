# Airline Passenger Satisfaction Analytics

Eine interaktive Streamlit-Anwendung zur Analyse und Vorhersage der Kundenzufriedenheit im Flugverkehr.

## Funktionen

- **Executive Dashboard**: Übersicht der wichtigsten KPIs und Zufriedenheitstrends
- **Detailed Analysis**: Tiefgehende Analyse von Servicebewertungen und Einflussfaktoren
- **Satisfaction Prediction**: Machine Learning-basierte Vorhersage der Kundenzufriedenheit
- **Customer Journey Map**: Visualisierung der Kundenerfahrung über verschiedene Touchpoints
- **Recommendations**: Datengestützte Empfehlungen zur Verbesserung der Kundenzufriedenheit

## Installation

```bash
pip install -r requirements.txt
```

## Ausführung

```bash
streamlit run capstone.py
```

## Datensatz

Die Anwendung verwendet den Datensatz `capstone_merged_data.csv`, der verschiedene Kundenbewertungen und Fluginformationen enthält.

## Modell

Die Anwendung nutzt ein vortrainiertes XGBoost-Modell (`xgb_model.sav`), um Kundenzufriedenheit zu prognostizieren.
